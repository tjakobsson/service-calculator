== Overview

The Toyota Servicekalkylator is a single-page static web application that calculates service intervals for Toyota vehicles based on annual driving distance. The user selects their yearly driving distance in mil (Swedish miles) via an interactive range slider and sees the corresponding service interval in days (primary), months, and weeks.

The application follows the "Precision Design" from the Stitch design system, featuring Toyota red branding (#bd0014), a hero section, value proposition cards, glassmorphism effects, and Material Symbols icons. It is entirely client-side with no backend, no database, and no external service dependencies. All logic, data, fonts, and icons are bundled into a static `dist/` directory. This satisfies constraint C-002 (fully offline operation).

== Technology Stack

* **ADR-0001:** Test-Driven Development — Vitest 4.1 + React Testing Library 16.3 (see docs/adrs/0001-adopt-tdd.yaml)
* **ADR-0002:** Vite 8, React 19, Tailwind CSS 4.2, Radix UI 1.4 (see docs/adrs/0002-frontend-stack.yaml)
* **Fonts:** Manrope (headlines) and Inter (body) via @fontsource, bundled locally (OFL licensed)
* **Icons:** Material Symbols Outlined via `material-symbols` npm package (Apache-2.0), bundled locally
* **Design:** Toyota Precision Design — Toyota red (#bd0014) primary, tonal surface layering, glassmorphism, no borders
* **Build output:** Static `dist/` directory — HTML, JS, CSS, font files, icon font. No runtime server required.
* **Skill tooling:** Python 3.14 with uv (pyyaml, jsonschema, jinja2) — used only for validation scripts, not part of the application.

== Cross-Cutting Concerns

=== Offline Operation (C-002)

All assets are bundled at build time. The application makes zero network requests at runtime. Fonts are loaded from local @fontsource packages. Material Symbols icons are loaded from the local material-symbols package. The korstracka.json lookup data is imported as a JS module during the Vite build.

=== Licensing (C-001)

All runtime dependencies use permissive licenses: React (MIT), Radix UI (MIT), Tailwind CSS (MIT), @fontsource packages (MIT with OFL font files), material-symbols (Apache-2.0). No copyleft dependencies.

=== Input Handling

The range slider constrains input to valid values (1000–5500 mil, step 100) at the UI level. The `lookupInterval` function receives pre-validated values directly from the slider. Results update live as the slider moves — no form submission required.
